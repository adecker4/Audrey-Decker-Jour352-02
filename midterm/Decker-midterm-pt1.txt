2.  Describe the purpose of a DIV tag.

At the most basic level, a DIV tag is a container. Its purpose is to group together HTML elements to format with css. It defines a section of the HTML document.

4.  There are two main "areas" or "sections" in the basic structure of an HTML page. In which area would you put a link to a stylesheet?

You put a link to the stylesheet in the <head> section of an HTML page, like so:
<head>
  <title>Audrey's Midterm</title>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>

10.  Why do we use media queries?

We use media queries to include a block of CSS only if a condition is true. The @media rule can be used to check certain things like width, height, etc.


11. What is the main difference between a serif and a sans-serif font?

The main difference between a serif and sans-serif font is that serif fonts have little lines at the ends of some letters/ characters while sans-serif do not have these lines.
